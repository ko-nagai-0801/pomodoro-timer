# Pomodoro Timer UX/Visual Design Specification

## Overview
macOS デスクトップに常駐するアウトラインのみのポモドーロタイマー。
背景完全透明、ストロークのみで描画し、作業の邪魔にならないミニマルデザイン。

---

## 1. サイズ・レイアウト

### ウィンドウ
- **ウィンドウサイズ**: 140x140 px
- **Canvas サイズ**: 140x140 px
- **背景**: 完全透明（後述のトリックで実現）

### 円形タイマー
- **中心座標**: (70, 70)
- **外径（ベースサークル）**: 半径 55px（直径 110px）
- **プログレスアーク**: 半径 55px（ベースサークルと同一パス上）
- **線幅（ベースサークル）**: 2px
- **線幅（プログレスアーク）**: 4px
- **パディング**: 外縁から 15px

### テキスト
- **残り時間（MM:SS）**: フォント `"Helvetica Neue"`, サイズ **22px**, bold
- **テキスト位置**: 中心 (70, 68) — 円の中心からわずかに上
- **モード表示テキスト**: フォント `"Helvetica Neue"`, サイズ **9px**, regular
- **モード表示位置**: (70, 90) — 時間テキストの下

---

## 2. カラースキーム

全て **アウトライン/ストロークのみ**。塗りつぶしなし。

### 作業モード (Focus)
| 要素 | カラーコード | 説明 |
|------|-------------|------|
| プログレスアーク | `#FF6B6B` | 暖かみのある赤。集中感を演出 |
| ベースサークル | `#4A4A4A` | ダークグレー。控えめなガイド |
| 時間テキスト | `#FF6B6B` | アークと同色で統一感 |
| モードテキスト | `#888888` | 控えめなグレー |

### 休憩モード (Break)
| 要素 | カラーコード | 説明 |
|------|-------------|------|
| プログレスアーク | `#4ECDC4` | ティールグリーン。リラックス感 |
| ベースサークル | `#4A4A4A` | 作業時と同じガイド |
| 時間テキスト | `#4ECDC4` | アークと同色 |
| モードテキスト | `#888888` | 控えめなグレー |

### 一時停止状態 (Paused)
| 要素 | カラーコード | 説明 |
|------|-------------|------|
| プログレスアーク | `#FFD93D` | アンバー。注意を引く黄色 |
| ベースサークル | `#4A4A4A` | 変化なし |
| 時間テキスト | `#FFD93D` | アークと同色 |

### 透明背景トリック（tkinter）
- **透明色**: `#F0F0F0`
- `root.wm_attributes("-transparentcolor", "#F0F0F0")` で透過実現
- Canvas の `bg` を `#F0F0F0` に設定
- **注意**: この色をUI要素に使わないこと

## 3. インタラクション設計

### クリック（左クリック / `<Button-1>`）
- **待機中**: タイマー開始（25:00 カウントダウン開始）
- **実行中**: 一時停止
- **一時停止中**: 再開
- **完了時**: 次のフェーズへ遷移（作業 → 休憩 or 休憩 → 作業）

### ドラッグ（`<B1-Motion>`）
- ウィンドウ移動
- `<ButtonPress-1>` でオフセット記録、`<B1-Motion>` でウィンドウ位置更新
- **判定**: Press から 5px 以上移動したらドラッグと判定し、クリックイベントを無効化

### 右クリック（`<Button-2>` macOS / `<Button-3>`）
- コンテキストメニュー表示:
  - `リセット` — タイマーを初期状態に戻す
  - `スキップ` — 現在のフェーズをスキップして次へ
  - `---`（セパレータ）
  - `作業: 25分 / 50分` — 作業時間切り替え
  - `休憩: 5分 / 10分` — 休憩時間切り替え
  - `---`（セパレータ）
  - `終了` — アプリケーション終了

### ホバー（`<Enter>` / `<Leave>`）
- **マウスイン時**: 追加情報をフェードイン表示（後述）
- **マウスアウト時**: 追加情報をフェードアウトで非表示
- フェード時間: 150ms

### ダブルクリック
- 使用しない（シングルクリックで全操作をカバー）

---

## 4. 表示要素

### 4.1 通常時（常に表示）

```
        ╭─── ベースサークル（#4A4A4A, 2px, 360度）
       ╱
  ┌──────────┐
  │    ╭──╮  │
  │   ╱    ╲ │ ← プログレスアーク（カラー, 4px, 残り角度）
  │  │25:00│ │ ← 残り時間テキスト（MM:SS）
  │   ╲    ╱ │
  │    ╰──╯  │
  └──────────┘
```

- **ベースサークル**: 360度の完全な円。点線ではなく実線（`dash` なし）。常に表示。
- **プログレスアーク**: 12時位置（90度）から時計回りに減少。`extent` は残り時間の割合 × 360度。`style=tk.ARC`。
- **残り時間テキスト**: `MM:SS` フォーマット。Canvas `create_text` で描画。

### 4.2 ホバー時（追加表示）

通常の要素に加え、以下を表示:

- **モードラベル**: 「集中」または「休憩」— 時間テキストの下 (70, 90)
- **ポモドーロカウンター**: 完了数をドットで表示
  - 位置: 円の下部 (70, 125)
  - 完了済み: 塗りつぶし円 `●` (`create_oval`, fill=カラー)
  - 未完了: アウトライン円 `○` (`create_oval`, outline=`#4A4A4A`)
  - ドットサイズ: 直径 6px
  - ドット間隔: 12px（中心間）
  - 最大4個表示（4ポモドーロで1セット）
- **ヒントテキスト**（待機時のみ）: 「クリックで開始」
  - 位置: (70, 90)
  - フォント: `"Helvetica Neue"`, 9px
  - カラー: `#888888`

### 4.3 状態遷移テーブル

| 状態 | 表示 | クリック動作 |
|------|------|-------------|
| idle（待機） | `25:00` + ベースサークルのみ | → running |
| running（実行中） | カウントダウン + プログレスアーク | → paused |
| paused（一時停止） | 時間固定 + アーク点滅 | → running |
| finished（完了） | `00:00` + フラッシュ | → 次フェーズの idle |

## 5. アニメーション

### 5.1 プログレスアーク
- **方向**: 12時位置（`start=90`）から**時計回り**に減少
- **更新間隔**: 1000ms（1秒ごと）
- **描画方法**: `canvas.itemconfig(arc_id, extent=-current_extent)`
  - `extent` は負の値で時計回りに描画
  - `current_extent = (remaining_seconds / total_seconds) * 360`

### 5.2 完了時フラッシュ
- タイマーが `00:00` に達した時に発動
- **効果**: プログレスアークとテキストの色を 3回点滅
- **パターン**: 表示(300ms) → 非表示(200ms) × 3回 = 合計 1500ms
- **非表示時**: アークとテキストを透明色 `#F0F0F0` に変更
- **点滅後**: 通常色に戻り `finished` 状態で待機

### 5.3 一時停止インジケーター
- **効果**: プログレスアークの `stipple` を切り替えて点滅
- **代替実装**: アークの透明度切り替え（色を `#F0F0F0` と元色で交互）
- **パターン**: 表示(800ms) → 非表示(400ms) の繰り返し
- `after()` で再帰的にスケジュール

### 5.4 ホバーフェードイン/アウト
- 追加要素の `state` を `tk.HIDDEN` / `tk.NORMAL` で切り替え
- 即時切り替え（tkinter で真のフェードは重いため）
- 切り替え時間: 即時（0ms）

## 6. ウィンドウ属性（実装メモ）

```python
# ウィンドウ基本設定
root.overrideredirect(True)          # タイトルバーなし
root.wm_attributes("-topmost", True) # 常に最前面
root.wm_attributes("-transparentcolor", "#F0F0F0")  # 透過色指定

# 初期位置: 画面右下から 20px マージン
screen_w = root.winfo_screenwidth()
screen_h = root.winfo_screenheight()
x = screen_w - 140 - 20
y = screen_h - 140 - 60  # Dock分の余白
root.geometry(f"140x140+{x}+{y}")
```
